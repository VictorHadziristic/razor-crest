Web Usability

What is Web Usability?​

"The usability of a website tells us how effectively, efficiently, and satisfactorily its visitors or users can see, or examine by other means, the website."

Source: https://health.usf.edu/is/blog/2018/03/13/what-is-website-usability--why-is-it-important

Why usability is important?​

The usability of a website is essentially part of the user's experience with a company. In essence, it is a part of customer service. A customer can go to a variety of companies to fulfill a need. As a competitor in the space, the user experience should be as fluid as possible, and in today's age, a good website is required to maintain this.

30 principles for good usability regarding web design.

Homepage

  1. A homepage should be attractive and draw users in.
  2. This page should be easy to navigate.
  3. This page should include all the most important links in your website
  4. A homepage should only make use of a limited amount of text and pictures, so as to not overwhelm users.
  5. Any planned changes for a website should be mentioned on this page

Navigation

  1. The back button in the browser should function as expected, to give users a reliable navigation tool.
  2. Links and buttons should behave consistently.
  3. Labels on tabs should be easy to understand.
  4. Sitemaps are a useful page to include to help users understand how to navigate a website.

Scrolling and Paging

  1. Avoid using horizontal scrolling.
  2. When there is a lot of text on a page, slower scrolling speeds should be used.

Content

  1. Make sure only necessary information is clearly displayed.
  2. Display information in a variety of ways – some visitors might like reading through info, others might like visuals or audio files.

Layout

  1. Pages should not be cluttered with extraneous information or images.
  2. Important items that you want your users to see right away should be in the top center of the page.
  3. Comparative information should always be on one page - don’t make users go back and forth between pages to compare the information.
  4. The most important information on a page should be on the top and the least on the bottom.
  5. Different items on a page should be aligned consistently.
  6. Be conservative with how much white space you include.

Links

  1. Text links are more useful than image links, so avoid using too many image links.
  2. Be consistent in how you alert a user as to which elements are clickable.
  3. Properly label internal and external links.
  4. Use appropriate link lengths.
  5. Let the user know if there are parts of an image they can click on for a link.

Headings, Titles, and Labels

  1. Headings, titles and labels should be descriptive.
  2. Highlight important headings visually, this will help readers find the information.
  3. Be concise in your headings and titles, but be sure to make them meaningful as well.

Accessibility

  1. Provide text for all images, animations, maps, and other media.
  2. Make sure the information on your page can be accessed without style sheets, which may interfere with assistive technology.
  3. Make sure that plugins and applets are available to everyone by testing them to be sure.
