Lab question responses

1. What's your modification? Any thoughts(why)? Steps?

For the first image on this page, I modified the image so that half of the car would be a different color. I used photoshop to accomplish this. I selected half of the image, placed it into a new layer, selected the colored part of the car using the selection tool and modified that selection using Photoshop's Hue/Saturation menu. I thought that making this modification along with the interactive component on this page would be a neat example to show off image modifications.

2. What are the differences you notice from the modification (e.g., file size change? Quality change?)

The image is originally a JPEG, which is a lossy format. Once I made my modifications, I exported the image as a PNG, a lossless format. The interactive component above loads either the orignial JPEG or the modified PNG, if you look closely, there really isn't a noticeable differnece in quality. This goes to show that there isn't much argument for lossless over lossy formats when it comes to web development. But the difference lies in the filesize, you might even notice that your first click on the component will cause a hiccup and refresh the page. This stems from the fact that the PNG is nearly 6 times the size of the JPEG version.

3. Is there any special consideration when you created/drew the new image(s)? What were the steps?

I thought I would make a logo for this example, and I wanted to demonstrate one of the characteristics of a particular format, namely, PNG. I drew up the logo in Illustrator, selected PNG as the output format and selected "transparent background". Now I could not really show off that the image indeed had a transparent background if it was sitting on a static white background. So I made a little front-end code so that when the user clicks on it, the CSS styling changes the background of the image, and this demonstrates that the image does indeed have an alpha channel, which is what controls pixel transparency.
